# Módulo: Banco de Dados com SQLite3

Este módulo introduz conceitos práticos de banco de dados relacionais e ensina passo a passo como executar as operações **CRUD** (Create, Read, Update, Delete) nativamente em Python, utilizando a biblioteca padrão `sqlite3`.

## Estrutura do Estudo:
- **Aula 01**: Introdução e Conexão com o banco
- **Aula 02**: Criando Tabelas (`CREATE`)
- **Aula 03**: Inserindo Dados (`INSERT`)
- **Aula 04**: Lendo Dados (`SELECT` - Read)
- **Aula 05**: Atualizando e Deletando (`UPDATE` e `DELETE`)
- **Aula 06**: Consultas Avançadas e Filtros
- **Aula 07**: Boas Práticas, Prevenção de SQL Injection e Gerenciadores de Contexto (`with`)

## Como Estudar
Siga a ordem sequencial dos arquivos `.ipynb`. Cada aula constrói conhecimento em cima da anterior. O banco de dados `meu_banco.db` será criado na sua pasta quando você rodar o primeiro código.

## Desafio Final
Ao concluir as aulas, abra o arquivo `Desafio_Final - Gerenciador de Tarefas.py`. Ele contém as instruções e uma estrutura em Python puro para você construir um sistema completo e colocar em prática todo o conhecimento de banco de dados!
